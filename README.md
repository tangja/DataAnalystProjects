# Data Analyst Projects

Below is an explanation for each file in this repository.

### data_cleaning.sql
A SQL file consisting of code for cleaning a dataset containing information on layoffs around the world from March 2020 to March 2023. Code was written to perform 4 tasks on the
dataset:

1. To remove duplicates that would otherwise make the data incorrect
2. To standardize the data and ensure that all data is consistent
3. To either populate or eliminate NULL/Blank values that would be unusable in finding trends
4. To remove any columns of data that would not be useful in finding trends

### data_analysis.sql
A SQL file that consisting of code for grouping and filtering different data groups that may be useful in finding in trends.

### excel_project_dataset.xlsx
An Excel file consisting of a dataset on bike sales, pivot tables created from the cleaned data, and a dashboard made up of the visualizations of the pivot tables.

### AirBnB_projects.twb
A Tableau file consisting of a dashboard with different visualizations for data trends based on a dataset of AirBnBs in Seattle in 2016.
